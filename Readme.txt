Para descargar el repositorio: 
1.	Ejecute CMD o Terminal si es Windows o Linux respectivamente
2.	Ubíquese en la carpeta de su preferencia.
3.	Ejecute el comando
git clone https://github.com/jorgealfonsobeltran/psp21.git  


Para cambiar los datos de la lista
1. Abra el archivo arreglo.txt
2. Modifiquelo agregando los valores de x_inicial-grados_de_libertad_-iteraciones-p_de_x-error_definido
de valores deben estar separadas por guiones (-)
3. Guarde el archivo y corra la aplicacion de nuevo


Para ver esta aplicacion en heroku
1. Vaya a la direccion https://labecos01-japsp21.herokuapp.com/

Para ver la documentación de este api vaya a ./psp21/target/site/apidocs 
Para ver la documentación de las pruebas este api vaya a ./psp21/target/site/testapidocs
